Erro ao carregar a ficha

Você parece estar usando um portfólio com fichas de uma versão antiga do Poder Supremo. Confira se o mesmo está em uma versão igual ou acima da 1.10.0 (em Ajuda → Versão)

Você não precisa fazer uma ficha nova do zero para poder usar o site: apenas abra o arquivo novamente e o salve com essa nova versão que automaticamente ele incluirá os dados necessários para carregá-lo aqui.
