Como Usar

Este site, Poder Supremo - Leitor de Fichas Online, tem a funcionalidade de permitir você carregar suas fichas para poder visualizá-las em qualquer lugar que estiver, permitindo várias formas de extração e visualização das mesmas, como também o tracker de combate, para fazer controle de combate entre vários personagens.

Uma vez que ele é um site companion para a aplicação original, você precisa dela para a criação das fichas. O Poder Supremo pode ser baixado no link a seguir:

• https://superbrego.itch.io/poder-supremo

Para começar tudo, primeiro carregue o portfólio clicando no botão do início da página.

Erro ao carregar a ficha

Você parece estar usando um portfólio com fichas de uma versão antiga do Poder Supremo. Confira se o mesmo está em uma versão igual ou acima da 1.11.3 (em Sobre → Versão)

Você não precisa fazer uma ficha nova do zero para poder usar o site: apenas abra o arquivo novamente e o salve com essa nova versão que automaticamente ele incluirá os dados necessários para carregá-lo aqui.
